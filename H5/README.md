# 基于vue多tab页实现
本模板提供了tab页面实现的源码，以及灵活控制tab的接口，简单易懂。

## tab页实现常见问题
* tab页面数据的缓存
* tab页面的滚动条状态
* tab页面的跳转调用以及刷新
* tab页面缓存手动清除，灵活控制。
* tab页面实现逻辑与业务逻辑分离，甚至取消tab，系统可以正常使用。
